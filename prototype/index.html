Figma Prototype

Videos

These videos show all of the parts of my prototype that I mocked up. It follows the same task format that I used in my user testing.

  1. Admin Functions: request admin access and change role groupings such that the 23s are considered alumni instead of current students
  2. Creating Chats as Admin: create a new public chat with only the 23s and 24s
  3. Event RSVP: RSVP yes to an Iftar event
The improvements that I made since my paper prototype are:
  1. More documentation: shows what the private/locked roles are, the difference between private and public chats, and that access to private roles must be requested
  2. The look of roles is consistent with filters/attributes that can be drag and dropped/clicked to be added and removed from different groupings. Similarly the look of the MSA/BSA buttons look more like filter buttons than buttons that take you to settings.
  3. Separated groupings from roles and made buttons more clear as to whether they refered to roles or groupings
